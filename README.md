# Spring Rest API

</div>

## To run

```shell
mvn spring-boot:run
```

## Features

* Persistence provided by PostgreSQL
* After run, swagger is available at -> localhost:8080/swagger-ui.html
* Create, read, update and delete.

## Notes

* Without validation.
